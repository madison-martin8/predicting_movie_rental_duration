# Predicting Movie Rental Durations

A machine learning project that aims to create the best model for predicting movie rental durations for a DVD rental company utilizing linear regression, random forest modelling, feature selection, and hyperparameter tuning techniques. 

# Installation Instructions
This project was completed in DataCamp's datalab environment using Python 3.8 

### Packages 
- sklearn 
- pandas 
- numpy
- matplotlib

### Data Source 
- DVD company rental dataset supplied by DataCamp within the project
